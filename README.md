# 2D Circular Quantum Well

https://parabul.github.io/QuantumWell/

A browser-based (WASM) simulation of a particle trapped in a 2D circular infinite potential well, demonstrating the Time-Dependent Schrödinger Equation (TDSE) via eigenstate expansion.

### Acknowledgments & Original Work
This project is a version of the educational software created by **Paul Falstad**.
